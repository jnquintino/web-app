## Passos iniciais

Para iniciar o projeto web devemos usar o seguinte comando:

```bash
npm install
# ou
yarn install
```

Depois digitamos o seguinte comando:

```bash
npm run dev
# or
yarn dev
```

Agora podemos acessar a aplicação em http://localhost:3000

## Pré-requisitos

Rodar a aplicação em Spring Boot que se encontra no repositório: https://github.com/jnquintino/hierarquia
